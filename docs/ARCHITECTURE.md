## 模块划分

### 1. 页面层

| 页面 | 路由 | 功能 |
|------|------|------|
| 首页 | `/` | 明星列表、最近更新、快捷操作 |
| 明星页 | `/star/:id` | 人物资料、关联内容、时间线 |
| 收藏库 | `/collection` | 全部收藏卡片、筛选、批量操作 |
| 时间线 | `/timeline` | 按时间/人物/平台查看轨迹 |
| 搜索 | `/search` | 全文搜索、多维度筛选 |
| 提醒 | `/reminder` | 新内容提醒、日历提醒 |
| 设置 | `/settings` | 导入导出、数据管理 |

### 2. 核心服务

#### 收藏管理（CollectionService）
- `add(url, person, tags)` — 收藏新内容
- `remove(id)` — 删除收藏
- `update(id, data)` — 更新内容属性
- `getByPerson(personId)` — 获取某人所有内容
- `getByPlatform(platform)` — 按平台筛选
- `markWatched(id)` / `markUnwatched(id)`
- `ignore(id)` / `block(id)`

#### 人物管理（PersonService）
- `create(name, aliases, keywords)` — 添加明星/团队
- `update(id, data)` — 更新资料
- `getTimeline(personId)` — 获取人物时间线
- `getStats(personId)` — 获取统计数据

#### 去重引擎（DedupEngine）
- `checkDuplicate(url, platform, title)` — 检查是否重复
- `getSimilar(title)` — 查找标题相似内容
- `isBlocked(url, platform, author)` — 检查是否被屏蔽

#### 时间线（TimelineService）
- `build(personId)` — 构建人物时间线
- `buildAll()` — 构建全局时间线
- `filter(options)` — 按条件筛选

#### 搜索（SearchService）
- `search(query, filters)` — 全文搜索
- `suggest(query)` — 搜索建议

#### 导入导出（BackupService）
- `exportJSON()` — 导出全部数据
- `importJSON(data)` — 导入数据
- `validateImport(data)` — 校验导入数据

## 数据流

```
用户操作 → Page → Service → Model → Storage
                                      ↓
                                  UI 更新 ← Store (Pinia)
```
